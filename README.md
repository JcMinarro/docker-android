docker-android
===================

manabugt/android:latest
https://registry.hub.docker.com/u/manabugt/android/

This is a Dockerfile for Android projects.

Included
----------

* Android SDK: r24.3.3
* Android NDK: r10e
* Build tools: 22.0.1
* Android API: 22
* Support maven repository
* Google maven repository
* Platform tools
* Make: 3.8.1
* Maven: 3.0.5
* oracle-java7
